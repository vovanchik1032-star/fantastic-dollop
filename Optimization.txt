Это список для оптимизации винды на малинке
1.Backup
Если ты хочешь сохранить именно настройки системы, чтобы потом «накатывать» её через тот же WoR или другой установщик.
​Инструмент: Dism++ (очень крутая утилита для ARM в том числе).
​Как делать:
​Запускаешь Dism++ прямо на Малинке (или подключив флешку к ПК).
​Выбираешь функцию «Восстановление» -> «Захват образа».
​Он создаст файл install.wim.
​Плюс: Этот файл будет весить мало (только реально занятое место). Его можно подсунуть в WoR вместо оригинального образа.
2.Сама оптимизация
​1. Отключение графических излишеств (SystemPropertiesPerformance)
​Это база, но на ARM это дает ощутимый буст FPS в интерфейсе.
​Нажми Win + R -> sysdm.cpl -> Дополнительно -> Параметры (Производительность).
​Выбери «Обеспечить наилучшее быстродействие».
Настрой файл подкачки (Твой «запасной» RAM)
​Без этого Майнкрафт даже не запустится.
​В том же окне «Быстродействие» перейди во вкладку Дополнительно -> Изменить.
​Сними галку «Автоматически выбирать объем...».
​Выбери диск C:, нажми «Указать размер».
​Поставь 3072 в оба поля (это 3 ГБ). Нажми «Задать» и «ОК».
​Перезагрузись.
​Лайфхак: Оставь галочку только на «Сглаживание неровностей экранных шрифтов», иначе глаза вытекут от пикселей в тексте.
​2. Убийство Defender и Телеметрии (через CMD)
​Если NTLite что-то оставил, эти команды добьют службы, которые постоянно сканируют диск и грузят CPU.
Открой CMD от администратора:
:: Полное отключение защитника (если он еще жив)
reg add "HKLM\SOFTWARE\Policies\Microsoft\Windows Defender" /v "DisableAntiSpyware" /t REG_DWORD /d 1 /f

:: Отключение телеметрии и сбора данных
reg add "HKLM\SOFTWARE\Policies\Microsoft\Windows\DataCollection" /v "AllowTelemetry" /t REG_DWORD /d 0 /f

:: Отключение фоновой проверки обновлений (чтобы не лагало во время игры)
sc config wuauserv start= disabled
3. Настройка электропитания (Максимальная мощь)
​Windows по умолчанию может пытаться «экономить» энергию, сбрасывая частоты ARM-процессора.
:: Включение схемы "Максимальная производительность"
powercfg /duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61
powercfg /setactive e9a42b02-d5df-448d-aa00-03f14749eb61
4. Очистка автозагрузки (Task Manager)
​Нажми Ctrl + Shift + Esc и во вкладке Автозагрузка отключи вообще всё. Даже если там один пункт — он ест память.
​5. Трюк с приоритетом для Minecraft
​Когда запустишь игру, не забудь сделать следующее:
​Сверни игру (Alt + Tab).
​Зайди в Диспетчер задач.
​Вкладка Подробности (Details).
​Найди процесс майнкрафта (или javaw.exe), правой кнопкой -> Задать приоритет -> Высокий.
Это заставит Windows отдавать все такты процессора игре, забивая на фоновые системные процессы.
​6. Удаление неиспользуемых драйверов (Ghostbuster)
​В Windows ARM часто висят драйверы для вещей, которых на Малинке нет (датчики приближения, мобильные модемы и т.д.).
​Можно использовать утилиту Ghostbuster, чтобы удалить «призрачные» устройства. Это освободит немного места в ядре системы.
​7. Сжатие папок через LZX (Advanced)
​Если места на SD-карте совсем мало, можно сжать папку с самой игрой алгоритмом, который почти не тратит CPU на распаковку:
compact /c /s /a /i /exe:lzx "C:\Путь_к_папке_с_игрой\*"
8. Отключи фоновую «грязь» (Службы)
​Открой CMD от имени администратора и вставь эти команды одной пачкой. Они выключат то, что Малинке вообще не нужно:
:: Выключаем поиск (индексация убивает SD-карту)
sc stop WSearch && sc config WSearch start= disabled

:: Выключаем обновление (чтобы не качало ничего в фоне)
sc stop wuauserv && sc config wuauserv start= disabled

:: Выключаем защитник (Defender — главный враг на 1ГБ ОЗУ)
reg add "HKLM\SOFTWARE\Policies\Microsoft\Windows Defender" /v DisableAntiSpyware /t REG_DWORD /d 1 /f

:: Выключаем эффекты прозрачности
reg add "HKCU\Software\Microsoft\Windows\CurrentVersion\Themes\Personalize" /v EnableTransparency /t REG_DWORD /d 0 /f
​9. Убей обои (Черный фон)
​Это звучит странно, но отрисовка красивой картинки в высоком разрешении занимает часть видеопамяти, которой у нас и так всего 1 ГБ (делится между CPU и GPU).
​Нажми правой кнопкой на рабочий стол -> Персонализация.
​Выбери Фон -> Сплошной цвет.
​Выбери Черный. Это самый «легкий» вариант для системы.
​10. Отключи эффекты прозрачности
​Прозрачность меню «Пуск» и окон заставляет видеочип пересчитывать пиксели под окнами.
​В том же меню Персонализация зайди в раздел Цвета.
​Выключи переключатель Эффекты прозрачности.
​11. Настройка драйвера в WoR (если еще можно)
​Если закачка дойдет и ты начнешь установку в WoR, обрати внимание на раздел Drivers.
​Если там есть выбор, используй минимальный набор драйверов. Чем меньше лишних .sys файлов загрузится в память, тем стабильнее будет стоять система.
12.powershell Get-mmAgent
Если в строке MemoryCompression стоит False, включи её:
powershell Enable-mmAgent -mc
Для Малинки с 1–2 ГБ ОЗУ это критически важно, так как процессор сожмет данные быстрее, чем они успеют улететь в медленный файл подкачки на SD-карте.
13: Настройка эмулятора ARM (Самое важное!)
​Так как игра будет эмулироваться, нужно принудительно заставить Windows использовать самый агрессивный режим производительности для этого процесса.
​Если игра уже скачана, найди её исполняемый файл GeometryDash.exe.
​Нажми по нему правой кнопкой мыши и выбери Свойства (Properties).
​Перейди во вкладку Совместимость (Compatibility).
​Нажми кнопку Изменить настройки параметров ARM (Change ARM emulation settings) внизу.
​В появившемся окне в выпадающем списке выбери режим Strict execution или Disable advanced optimizations (в разных сборках называется по-разному, нам нужно отключить динамическую проверку кода, чтобы сэкономить ресурсы процессора). Нажми «Применить».
​14: Включение аппаратного ускорения и графического режима
​Нужно отдать всю графику на съедение видеочипу Broadcom малинки, разгрузив центральный процессор.
reg add "HKCU\Software\Microsoft\DirectX\UserGpuPreferences" /v "GeometryDash.exe" /t REG_SZ /d "GpuPreference=2;" /f
